#Smart Diet Meal Planner
The Smart Diet Meal Planner is a web-based application that helps users generate personalized diet plans based on their health goals and individual inputs. The system uses predefined logic along with a machine learning model to recommend three daily meals for users. It provides an easy-to-use and responsive interface built with React, while backend services are handled using Node.js. User data and meal information are stored and managed using MongoDB.
